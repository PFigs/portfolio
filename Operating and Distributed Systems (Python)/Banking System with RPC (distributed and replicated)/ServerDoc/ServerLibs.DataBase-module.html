Module DataBase

source code

This module includes classes and methods for the management and updating of the database. Several aspects were taken into account such as concurrent use by multiple threads using the block and other critical areas.

A database server has associated with it two objects: one is responsible for lock management in critical areas and the other by the deposit of the amount of interest in each account. For the second part of the project is assumed that the database can be replicated across multiple machines and only one of them has these objects in execution.
